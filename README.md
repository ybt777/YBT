# YBT 自动售卖

这是一个基于 Python 的自动售卖辅助工具。当前仓库保留了旧版固定流程，同时新增了 `ui3.0_action_builder.py`：一个可以自定义鼠标点击、键盘按键、文字输入和等待步骤的动作编排版本。

## 本次更新

- 新增 `自动售卖 3.0 - 自定义动作版`。
- 将原来写死在代码里的 `right`、`backspace`、`enter`、A/B/C 三次点击，改成可编辑的动作列表。
- 支持添加、删除、上移、下移、选中修改动作。
- 支持动作类型：鼠标点击、键盘按键、输入文字、等待。
- 支持保存和加载方案，配置文件为同目录下的 `action_config.json`。
- 支持从旧版 `config.json` 导入 A/B/C 坐标和延迟，自动生成旧版等价流程。
- 坐标读取已改为“读取上次点击坐标”：先在目标位置真实点击一次，再回到软件读取，避免误以为读取的是鼠标悬停位置。

## 文件说明

- `自动售卖/ui3.0_action_builder.py`：最新版自定义动作编排界面。
- `自动售卖/ui2.0.py`：旧版三坐标 UI。
- `自动售卖/ui3.0_action_builder.spec`：3.0 文件夹模式打包配置。

## 源码运行

进入项目目录：

```powershell
cd D:\YBT_HOME\desktop\project-1.0\自动售卖
python ui3.0_action_builder.py
```

如果使用 PyCharm，直接打开 `自动售卖/ui3.0_action_builder.py` 运行即可。

## 使用教程

1. 打开 `自动售卖 3.0 - 自定义动作版`。
2. 设置启动键和停止键，默认启动键为 `F4`，停止键为 `F12`。
3. 添加动作：
   - 鼠标点击：先在目标位置真实点击一次，再回软件点击“读取上次点击坐标”，然后添加动作。
   - 键盘按键：选择要按下的键，例如 `right`、`backspace`、`enter`。
   - 输入文字：填写要输入的内容。
   - 等待：填写等待秒数。
4. 在动作列表中可以选中、更新、删除、上移、下移动作。
5. 点击“保存方案”保存当前动作列表。
6. 点击“启动热键监听”，之后按启动键执行一次动作序列，按停止键请求停止。
7. 如果要沿用旧版售卖流程，可以点击“导入旧版售卖流程”。程序会优先读取旧版 `config.json` 中的 A/B/C 坐标。

## 打包说明

项目使用 PyInstaller 的文件夹模式打包。文件夹模式会生成一个包含 exe 和依赖文件的目录，比单文件模式启动更稳，也更容易排查问题。

示例命令：

```powershell
cd D:\YBT_HOME\desktop\project-1.0\自动售卖
..\.venv\Scripts\pyinstaller.exe --noconfirm --onedir --windowed --name "YBT-auto-sell-3.0-action-builder" ui3.0_action_builder.py
```

生成后的文件夹可以整体压缩后放入 GitHub Release 的 assets。

## 注意事项

- 本工具会模拟鼠标和键盘输入，使用前请先确认目标窗口和坐标正确。
- `pyautogui.FAILSAFE` 已开启，把鼠标快速移到屏幕左上角通常可以触发安全停止。
- 建议先用测试环境或非关键页面验证动作序列，再用于正式流程。
